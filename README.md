SearCHemicals
=============

SearCHemicals is a simple IUPAC structure lookup app.
------------------------

Key features:
•  picture of skeletal formula

•  IUPAC name

•  search history

•  favorite molecules

•  stereochemistry

•  different structural formula options

•  share molecule
